# 📈 Stock Trend Predictor (ML + Technical Indicators)

This project is a Streamlit web app that predicts short-term stock trends using Machine Learning and technical indicators like Moving Averages and RSI.

---

## 🚀 Features

- 📊 Live stock data using yfinance  
- 🤖 Machine Learning model (Random Forest)  
- 📈 Technical indicators:
  - Moving Averages (MA10, MA50)
  - RSI (Relative Strength Index)  
- 📉 Trend prediction (UP / DOWN)  
- 📊 Model accuracy display  
- 💡 Trading insights (Overbought / Oversold signals)  

---

## 🛠️ Tech Stack

- Python  
- Pandas, NumPy  
- Scikit-learn  
- Streamlit  
- yfinance  

---

## ▶️ How to Run Locally

### 1. Clone the repo
```bash
git clone https://github.com/your-username/stock-ml-app.git
cd stock-ml-app
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Run the app
```bash
streamlit run app.py
```
